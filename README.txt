AI PORTFOLIO SHOWCASE


------------------------------------------------------------

OVERVIEW

Welcome to my AI Portfolio Project, a collection of machine learning and AI-powered applications demonstrating end-to-end data science workflows—from data preprocessing, model training, to deployment and user interaction.

This project is designed as a central AI showcase platform with a main menu interface that allows users to access multiple AI applications in one place.

------------------------------------------------------------

MAIN INTERFACE

The project is launched through:

app.py

This acts as the main menu dashboard, allowing navigation to all AI modules.

------------------------------------------------------------

AI PROJECTS INCLUDED

AI Chatbot:
An intelligent conversational chatbot powered by natural language processing.

Diabetes Prediction Model:
A machine learning classification model that predicts diabetes risk based on medical inputs.

House Price Prediction Model:
A regression model that estimates property prices using housing features.

Plant Disease Detection System:
A computer vision model that detects plant diseases from images for agricultural assistance.

INSTALLATION & SETUP

1. Clone repo:
git clone https://github.com/mdrhernandez963-oss/Employment_AI-ML_Portfolio
cd ai-portfolio

2. Create environment:
python -m venv venv

Activate (Windows):
venv\Scripts\activate

3. Install dependencies:
pip install -r requirements.txt

------------------------------------------------------------

RUN APPLICATION

streamlit run app.py
